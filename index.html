Math Handouts

A few of my math handouts and slides are included here, and I hope to amass a large collection throughout the years. Most will in preparation for Canadian (i.e. COMC, C(J)MO and Euclid) and MAA Contests (i.e. AMC 8, AMC 10, AIME, etc.) Hope you enjoy! You may have noticed that the handouts are divided into categories. Here is a brief summary:
  1. Level 1: For Beginner to Late AMC 10/12
  2. Level 2: For AIME (you could consider this "Pre-Olympiad")
  3. Level 3: For C(J)MO/USA(J)MO (Olympiad Level)

Level 1 Handouts

These slides are from lectures I have given at my school's math circle.
  1. Introductory Counting 1 (

    2. An introductory lecture to Combinatorics (including ratios and proportions, SFFT, etc.).
  2. Introductory Counting 2 (

    )
    3. Introduction to the n choose k principle and counting when order doesn't matter
  3. Introductory Counting 3 (
    PDF Version

    4. Committee Forming Problems with an emphasis on counting objects when order doesn't matter.
  4. Probability (
    PDF Version
    )
    5. A basic lecture on probability and its uses in competition problems. Some set theory notation is included as well.
  5. Introductory Number Theory (
    PDF Version
    )
    6. A introductory lecture on number theory which covers number theoretic functions (eg. floor, ceiling and divisor counting functions), GCD and LCM, and primes.
  6. Modular Arithmetic (
    PDF Version
    )
    7. A lecture on modular arithmetic operations and uses in math contests.
  7. Modular Arithmetic Problems (
    PDF Version
    )
    8. A lecture focused on problems solving using various modular arithmetic operations.
  8. Functions and Chinese Remainder Theorem (
    PDF Version
    )
    9. A more advanced lecture on types of functions-injective,surjective and bijective-treated non rigorously. The Chinese Remainder theorem is also introduced to solve some contest problems.
